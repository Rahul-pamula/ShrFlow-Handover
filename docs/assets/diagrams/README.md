# Diagrams storage

Store source diagrams (Mermaid, draw.io, or SVG source) here and export PNG/SVG for inclusion in docs. Keep an editable source file alongside rendered exports.

Suggested structure:
- `source/` — editable mermaid/.mmd or drawio files
- `exports/` — generated PNG/SVG for docs site

When updating architecture, update the diagram source and regenerate exports.
